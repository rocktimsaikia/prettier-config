# @rocktimsaikia/prettier-config

> My personal Prettier config
<br/>

## Install

```sh
pnpm add -D @rocktimsaikia/prettier-config
```
<br/>

## Usage

In `package.json`:

```json
{
  "prettier": "@egoist/prettier-config"
}
```
